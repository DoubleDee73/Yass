Import Melody

You can create your own karaoke version for a given MP3 file.

Finding correct tones is the hardest task. The process is much easier if you can extract the melody from a matching KAR file.

A KAR is a specific MIDI file with synchronized lyrics. If you only have a MIDI file, you can still extract the melody, but have to synchronize the songtext to the music on your own.

Please choose a KAR or MIDI file.
