Melodie importieren

Sie können zu einer gegebenen MP3-Datei Ihre eigene Karaokeversion erstellen.

Dabei ist die Bestimmung der korrekten Position und Höhe der Noten die schwierigste Aufgabe. Es wird um einiges einfacher, wenn Sie die Melodie aus einer passenden KAR-Datei extrahieren können.

Eine KAR-Datei ist eine spezielle MIDI-Datei mit synchronisiertem Songtext. Falls Sie nur eine MIDI-Datei besitzen, können Sie zwar immerhin schon die Melodie extrahieren, müssen aber den Songtext selbst zur Musik synchronisieren.

Bitte wählen Sie eine KAR- oder MIDI-Datei.
