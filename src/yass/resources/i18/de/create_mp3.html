Neuen Song erstellen

Nun wird eine Audiodatei (MP3) des Songs benötigt. Sie sollte mit konstanter Bitrate kodiert sein (CBR); falls nicht, muss sie neu kodiert werden. Eine Bitrate von 192bit oder weniger genügt, bei Joint Stereo. Die Datei wird in den Songordner kopiert.
