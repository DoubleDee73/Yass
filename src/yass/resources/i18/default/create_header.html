Header Information

Default information is taken from the MP3 filename (Artist - Title) and the ID3 tags. The BPM is set to the maximum tempo of your KAR/MIDI file (or to 300, if not available). Do not change that value unless you know what you are doing.

Expert information: Later, you can multiply or divide the BPM by pressing CTRL-M/D; note length will be adjusted. You should choose a multiple close to 300 for performance reasons.

Please check and correct this data.
