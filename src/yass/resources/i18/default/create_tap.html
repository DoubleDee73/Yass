Finetuning

Your song file is about to be created. You can then proceed as follows.
  1. Approximate notes: You can quickly record position and length by tapping to the music.
  2. Finetune them: Each note must be checked and corrected manually.
Finetuning takes about 2-3 hours for an expert. Be patient: you will learn how to hear correct note lengths and tones.

Have fun!
