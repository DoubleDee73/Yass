Créer une nouvelle chanson

Veuillez maintenant spécifier votre fichier MP3. Assurez-vous qu’il est encodé avec un débit binaire constant (CBR) ; sinon, veuillez le réencoder. Choisissez un débit de 192 kbit/s ou moins, et le mode joint stéréo. Le fichier sera copié dans le répertoire des chansons.
